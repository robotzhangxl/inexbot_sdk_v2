# 总线

总线通讯协议配置，包括 CANopen、ENI 文件和从站管理。

---

## CANopen

### 上位机查询 CAN 支持的功能

**命令字：** `0x7000`

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| canPort | int | 是 | CAN 端口号 |
| function | int | 是 | 功能类型：`0`=标准帧，`1`=扩展帧，`2`=远程帧，`3`=过滤 ID |

### 请求示例

```json
{
  "canPort": 1,
  "function": 0
}
```

### 响应参数

| 参数 | 类型 | 说明 |
|------|------|------|
| canPort | int | CAN 端口号 |
| support | bool | 硬件是否支持该功能 |

---

### 上位机设置 CAN 参数

**命令字：** `0x7002`

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| canPort | int | 是 | CAN 端口号 |
| canBaud | int | 是 | CAN 通讯波特率 |
| frameFormat | int | 否 | 帧格式：`0`=标准帧，`1`=扩展帧 |
| enableRecvFiltert | bool | 否 | 是否开启过滤 ID |
| recvFilterId | int | 否 | 过滤 ID |

### 请求示例

```json
{
  "canPort": 1,
  "canBaud": 115200,
  "frameFormat": 0,
  "enableRecvFiltert": true,
  "recvFilterId": 1
}
```

---

### 上位机请求接收一帧 CAN 数据

**命令字：** `0x7003`

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| canPort | int | 是 | CAN 端口号 |

### 响应参数

| 参数 | 类型 | 说明 |
|------|------|------|
| canPort | int | CAN 端口号 |
| data | int[] | 接收到的数据 |
| Length | int | 数据长度 |
| recvCanID | int | 接收到的帧 ID |

---

### 上位机请求发送一帧 CAN 数据

**命令字：** `0x7005`

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| canPort | int | 是 | CAN 端口号 |
| data | int[] | 发送的数据 |
| sendCanID | int | 发送帧的 ID |
| sendLen | int | 数据长度，范围 [1, 8] |
| useRemoteFrame | bool | 是否使用远程帧 |

### 请求示例

```json
{
  "canPort": 1,
  "data": [10, 20, 30],
  "sendCanID": 0x123,
  "sendLen": 3,
  "useRemoteFrame": false
}
```

## ENI 文件

### 上位机查询 ENI 文件列表

**命令字：** `0x7010`

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| num | int | 是 | 获取数目：`1`=当前使用文件，`10`=所有文件 |

### 响应参数

| 参数 | 类型 | 说明 |
|------|------|------|
| absolutepath | string | ENI 文件所在文件夹 |
| ENIfilelist | string[] | ENI 文件列表 |
| ENIfilenum | int | ENI 文件数目 |

---

### 上位机查询 ENI 参数

**命令字：** `0x7012`

### 响应参数

| 参数 | 类型 | 说明 |
|------|------|------|
| ENIName | string | ENI 文件名称 |
| isHaveENI | int | 识别结果：`-1`=无 ENI，`0`=未识别，`1`=已识别 |

---

## 从站列表

### 上位机查询从站列表

**命令字：** `0x7020`

### 响应参数

| 参数 | 类型 | 说明 |
|------|------|------|
| IONum | int[] | IO 从站列表 |
| servoNum | int[] | 伺服从站列表 |
| slaveType | string[] | 从站中文名称列表 |
| slaveTypeEnglish | string[] | 从站英文名称列表 |

### 响应示例

```json
{
  "IONum": [0, 0, 0, 0, 0, 0],
  "servoNum": [1, 2, 3, 4, 5, 6],
  "slaveType": ["禾川X3E", "禾川X3E", "禾川X3E", "禾川X3E", "禾川X3E", "禾川X3E"],
  "slaveTypeEnglish": ["hcfa_x3e", "hcfa_x3e", "hcfa_x3e", "hcfa_x3e", "hcfa_x3e", "hcfa_x3e"]
}
```
