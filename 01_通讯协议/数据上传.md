# 数据上传

控制器数据采集与 FTP 上传协议配置。

---

## 上位机设置数据传输开关

**命令字：** `0x7330`

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| robot | int | 是 | 机器人号，范围 [1, 4] |
| enable | bool | 是 | 数据传输开关 |

### 请求示例

```json
{
  "robot": 1,
  "enable": true
}
```

---

## 上位机设置数据上传参数

**命令字：** `0x7333`

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| robot | int | 是 | 机器人号，范围 [1, 4] |
| serverIP | string | 是 | 服务器 IP |
| serverPort | int | 是 | 端口号，范围 (0, 65535] |
| username | string | 否 | 用户名 |
| password | string | 否 | 密码 |
| filepath | string | 否 | 文件路径 |
| fileSuffix | string | 否 | 文件格式：`csv` 或 `txt` |
| uploadMode | string | 否 | 上传方式，默认 `FTP` |
| dataCollectTime | float | 否 | 数据采集周期（s），范围 [0, 999999] |
| dataUploadTime | float | 否 | 数据上传周期（s），范围 [0, 999999] |
| sendInfoFileFlag | bool | 否 | 是否发送说明文档 |

### 请求示例

```json
{
  "robot": 1,
  "serverIP": "192.168.1.14",
  "serverPort": 21,
  "username": "admin",
  "password": "",
  "filepath": "",
  "fileSuffix": "csv",
  "uploadMode": "FTP",
  "dataCollectTime": 1.0,
  "dataUploadTime": 30.0,
  "sendInfoFileFlag": false
}
```

## 上位机设置数据格式

**命令字：** `0x7336`

> 数据格式设置与查询分四次进行

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| robot | int | 是 | 机器人号，范围 [1, 4] |
| type | int | 是 | 格式类型：`1`=说明文档文件名，`2`=说明文档文件内容，`3`=数据文档文件名，`4`=数据文档文件内容 |
| format | string | 是 | 数据格式，如 `Robot-R1_$Y%-$m%-$d%_$H%-$M%-$S%` |

### 请求示例

```json
{
  "robot": 1,
  "type": 1,
  "format": "Robot-R1_$Y%-$m%-$d%_$H%-$M%-$S%"
}
```

### 格式说明

| 占位符 | 含义 |
|--------|------|
| $Y% | 年 |
| $m% | 月 |
| $d% | 日 |
| $H% | 时 |
| $M% | 分 |
| $S% | 秒 |
