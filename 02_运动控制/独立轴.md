# 独立轴

独立轴参数配置与控制。

---

## 独立轴参数

### 上位机设置独立轴界面参数

**命令字：** `0x2800`

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| axisNum | int | 是 | 轴编号 |
| isTrack | bool | 否 | 是否是地轨 |
| motorDir | int | 否 | 电机方向：`1`=正，`-1`=负 |
| encoderBitLineditValue | double | 否 | 编码器位数 |
| reductionRatioValue | double | 否 | 减速比 |
| positiveLimitValue | double | 否 | 正限位 |
| inverseLimitValue | double | 否 | 反限位 |
| ratedPositiveSpeedValue | double | 否 | 额定正转速（rpm） |
| ratedReverseSpeedValue | double | 否 | 额定反转速（rpm） |
| maximumPositiveSpeedValue | double | 否 | 最大正转速（倍率） |
| maximumReverseSpeedValue | double | 否 | 最大反转速（倍率） |
| maxAccValue | double | 否 | 最大加速度（速度倍率） |
| maxDecValue | double | 否 | 最大减速度（速度倍率） |
| angularDistanceConversionRatioValue | double | 否 | 转换比 |
| selectBindServo | uint | 否 | 绑定伺服编号 |

---

### 上位机查询独立轴界面参数

**命令字：** `0x2801`

### 上位机删除独立轴参数

**命令字：** `0x2805`

---

## 独立轴位置

### 上位机查询独立轴位置

**命令字：** `0x2806`

### 响应参数

| 参数 | 类型 | 说明 |
|------|------|------|
| currentPos | double | 当前位置 |

---

### 独立轴零点标定

**命令字：** `0x2808`

---

## 轴独立控制

### 独立控制轴 PV 运动

**命令字：** `0x30E0`

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| robot | int | 是 | 机器人编号，范围 [1, 4] |
| axisNum | int | 是 | 轴编号 |
| vel | int | 是 | 轴速度（rpm），范围 [-6000, 6000] |
| dir | int | 是 | 运动方向：`1`=正，`-1`=反向 |
| acc | double | 否 | 加速度（速度倍率） |
| dec | double | 否 | 减速度（速度倍率） |

---

### 独立控制轴 PV 停止

**命令字：** `0x30E1`

### 取消轴独立控制运动

**命令字：** `0x30E4`

---

### 独立轴回零

**命令字：** `0x30E5`

### 独立轴回零停止

**命令字：** `0x30E6`

---

### 独立轴点动

**命令字：** `0x30E7`

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| axisNum | int | 是 | 独立轴编号 |
| vel | int | 是 | 速度（rpm） |
| acc | double | 否 | 加速度 |
| dec | double | 否 | 减速度 |
| dir | int | 是 | 运动方向：`1`=正，`-1`=反向 |
