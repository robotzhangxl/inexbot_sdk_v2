# 运动控制

机器人伺服状态、操作模式、示教模式、使能、点动和运动指令。

---

## 伺服状态

### 上位机设置机器人伺服状态

**命令字：** `0x3000`

> 当前伺服状态为「伺服错误」时，无法通过此命令修改伺服状态。

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| robot | int | 是 | 当前机器人号，范围 [1, 4] |
| state | int | 是 | 伺服状态：`0`=伺服停止，`1`=伺服就绪 |

### 请求示例

```json
{
  "robot": 1,
  "state": 1
}
```

---

### 上位机查询机器人伺服状态

**命令字：** `0x3001`

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| robot | int | 是 | 当前机器人号，范围 [1, 4] |

### 响应参数

| 参数 | 类型 | 说明 |
|------|------|------|
| robot | int | 当前机器人号 |
| state | int | 伺服状态：`0`=停止，`1`=就绪，`2`=错误，`3`=运行 |
| mode | int | 模式：`0`=单机模式，`1`=多机模式 |

---

### 上位机查询机器人伺服连接方式

**命令字：** `0x3004`

### 响应参数

| 参数 | 类型 | 说明 |
|------|------|------|
| robot | int | 当前机器人号 |
| servoType | int | 连接方式：`0`=实际伺服，`1`=虚拟伺服 |

---

## 示教模式

### 上位机设置当前示教模式类型

**命令字：** `0x3020`

> 拖拽模式运动时切换为点动模式会失败。

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| teachType | int | 是 | 示教模式类型：`0`=点动（单动），`1`=拖拽示教，`2`=点动（联动） |

### 请求示例

```json
{
  "teachType": 0
}
```

---

### 上位机查询当前示教模式类型

**命令字：** `0x3021`

### 响应参数

| 参数 | 类型 | 说明 |

## 使能

### 上位机设置使能状态

**命令字：** `0x3030`

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| deadman | int | 是 | 上下使能开关：`0`=下使能，`1`=上使能 |
| deadmanmode | int | 否 | 触发方式：默认 `0`（软件触发） |

> 控制系统上使能后，可以进行点动操作。

### 请求示例

```json
{
  "deadman": 1,
  "deadmanmode": 0
}
```

---

### 上位机查询使能状态

**命令字：** `0x3031`

### 响应参数

| 参数 | 类型 | 说明 |
|------|------|------|
| deadman | int | 当前使能状态 |

---

### 上位机设置机器人使能触发模式

**命令字：** `0x3033`

> 硬件触发模式下，上下使能端口不能同时为 0。

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| robot | int | 是 | 当前机器人号，范围 [1, 4] |
| mode | int | 是 | 触发模式：`0`=软件触发，`1`=硬件触发 |
| deadmanPortOne | int | 是 | 上使能端口（范围：0 ~ 已连接 IO 端口总数） |
| deadmanPortTwo | int | 是 | 下使能端口（范围：0 ~ 已连接 IO 端口总数） |

### 请求示例

```json
{
  "robot": 1,
  "mode": 0,
  "deadmanPortOne": 14,
  "deadmanPortTwo": 2
}
```

---

## 机器人速度

### 上位机请求设置机器人速度参数

**命令字：** `0x3050`

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| robot | int | 是 | 当前机器人号 |
| type | int | 是 | 速度模式：`0`=点动速度，`1`=关节坐标定距移动，`2`=直角坐标定距移动 |
| speed | int | 是 | 全局速度，范围 [1, 100]% |
| mocroDotSpeedACS | double | 否 | 关节坐标定距移动速度（°），范围 (0, 2000] |
| mocroDotSpeedMCS | double | 否 | 直角坐标定距移动速度（mm），范围 (0, 2000] |

### 请求示例

```json
{
  "robot": 1,
  "type": 0,
  "speed": 5,
  "mocroDotSpeedACS": 0.1,
  "mocroDotSpeedMCS": 0.1
}
```

---

### 上位机查询机器人速度参数

**命令字：** `0x3051`

### 响应参数

> 同设置参数

## 运动到点

### 上位机请求机器人回零

**命令字：** `0x3070`

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| robot | int | 是 | 当前机器人号 |
| isWithExternal | int | 否 | 是否包含外部轴：`0`=否，`1`=是 |

> **前提条件：** 运动前需将伺服设置为使能状态。

### 请求示例

```json
{
  "robot": 1,
  "isWithExternal": 0
}
```
