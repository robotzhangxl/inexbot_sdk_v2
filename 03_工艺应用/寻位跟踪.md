# 寻位跟踪


## 跟踪

### 命令字：0x8740

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| "fileNum" | int | 跟踪文件号；int类型；取值范围[1,99] |
| "robot" | int | 机器人号；int类型；取值范围[1,4] |
| "sensorType" | int | 跟踪类型;int类型；取值范围[0,2]；0:线激光；1:电弧；2:弧压 |

```json
{
"fileNum":1,
"robot":1,
"sensorType":0
}
```

### 命令字：0x8750

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| "craftid" | int | 跟踪文件号；int类型；取值范围[1,99] |
| "lateralCompensation" | object |  |
| "accFactor" | string | 补偿加速度倍数；double类型；取值范围[0.1,10]；对应当前运动指令加速 |
| "algorithmType" | int | 偏差提取类型；int类型；取值范围[0,1]；0:均值；1:手填值 |
| "beginCycleNum" | int | 开始采样周期数；int类型；取值范围[1,100] |

| "maxSingleLen" | double | 每次最大补偿量；double类型；取值范围[0,10] |
| "scale" | double | 比例系数；double类型；取值范围[0,1] |
| "switchon" | bool | 补偿开关；bool类型；true：打开，false：关闭 |
| "robot" | int | 机器人号；int类型；取值范围[1,4] |

```json
{

}
```
