### 设置重叠模式参数

### 命令字：0x8050

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| "robot" | int | 当前机器人；int类型；取值范围：[1,4] |
| "craftID" | int | 工艺号；int类型 |
| "pallet" | object | "floorSum" : 1；层数；int类型；取值范围：[1,999]；"overlapType" : 0,；重复关系；int类型；0：相同；1： 交替；2：自定义；"layHeightOffset" : 0.0；放货点高度补偿；double类型 |
| "fixedLayHeight" | bool | 固定放货点高度；bool类型；true：是；false：否 |
| "fixedSupHeight" | bool | 固定辅助点高度；bool类型；true：是；false：否 |
| "columnLay" | bool | 竖直方向排列；bool类型；true：是；false：否 |
| "floorAutoJustified" | string | 层自动对齐；bool类型；true：是；false：否 |
| "poseAutorotation" | string | 姿态自动旋转；bool类型；true：是；false：否 |
| "fixedEnterPoint" | string | 固定入口点位置；bool类型；true：是；false：否 |
| "graphicNum" | array | 图形编号；int类型；取值范围：[0,9] |
| "heightRevise" | array | 高度修正；double类型 |

```json
{
"robot":1,
"craftID":1,
"pallet":
{
"floorSum" : 1,
"overlapType" : 0,   //0：相同，1： 交替，2：自定义
"layHeightOffset" : 0.0,      //放货点高度补偿
"fixedLayHeight":false,      //固定放货点高度
"fixedSupHeight":false,      //固定辅助点高度
"columnLay":false,         //竖直方向排列
"floorAutoJustified":false,      //层自动对齐
"poseAutorotation":false,   //姿态自动旋转
"fixedEnterPoint":false,     //固定入口点位置
"graphicNum":[1,2,1,2]，      //图形编号
"heightRevise":[1.1,-2.2]      //高度修正
}
}
```
