# 报错
### 清错
说明：
在清除伺服错误时使用
上位机清除伺服错误
命令字：0x1100
"robot":1
表示要操作的机器人编号；int类型
取值范围[1,4]
```json
{
"robot":1,
}
```

清除伺服错误后，返回清除结果
命令字：0x1101
"robot": 1
表示机器人编号；int类型
取值范围[1,4]
"clearErrorFlag": true
是否清除成功；bool类型

```json
{
"robot":1,
"clearErrorFlag":true,
}
```
### 信息提示
说明：
控制器向上位机发送错误提示/警告提示/消息提示/弹窗提示
# 错误提示
命令字：0x1110
"robot"0
表示机器人编号； int类型
取值范围[0,4]，等于0时表示未选择机器人类型或者错误内容与机器人无关
"kind":2
弹窗提示种类；int类型
0为消息，1为警告，2为报错
"data":"指令[0x5743]不存在"
消息条内容；string类型
以上信息为一个示例，不同的错误对应不同的内容
"code": 28680;
消息编码；int类型
"param": [22339,3]
消息内容参数；int类型
根据具体编码的不同，会有不同个数，一共预留5个
```json
{
"data":"指令[0x5743]不存在",
"kind":2,
}
```
# 警告提示
命令字：0x1111
"robot":1
表示机器人编号；int类型
取值范围[0,4]，等于0时表示未选择机器人类型或者警告内容与机器人无关
"kind": 1
弹窗提示种类；int类型
0为消息，1为警告，2为报错
"data": "机器人1参数配置文件不存在，已创建，请配置参数"
消息条内容；string类型
以上信息为一个示例，不同的警告对应不同的内容
"code": 16385
消息编码；int类型
"param":  [1]
消息内容参数；int类型
根据具体编码的不同，会有不同个数，一共预留5个
```json
{
"data":"机器人1参数配置文件不存在，已创建，请配置参数",
"kind":1,
}
```
# 消息提示
命令字：0x1112
"robot": 0
表示机器人编号； int类型
取值范围[0,4]，等于0时表示未选择机器人类型或者消息内容与机器人无关。
"kind": 0
弹窗提示种类；int类型
0为消息，1为警告，2为报错
"data": "自动备份成功"
消息条内容；string类型
以上信息为一个示例，不同的消息对应不同的内容
"code": 21764
消息编码；int类型
```json
{
"data":"自动备份成功",
"kind":0,
}
```
# 弹窗提示
命令字：0x1113
"count": 1
弹窗按钮数量； int类型
最大为3
"text1": "xxx"
弹窗居中显示的主文本；string类型
"text2": "xxx"
按钮1的文本；string类型
"text3": "xxx"
按钮2的文本；string类型
"text4": "xxx"
按钮3的文本；string类型
""kind": 1
弹窗提示种类；int类型
0为消息，1为警告，2为报错
"node": 1
机器人序号；int类型
取值范围[1,4]
"Tips": true
弹窗呼出；bool类型
弹窗呼出为true
```json
{
"text1":"示例"
"text2":"示例"
"text3":"示例"
"text4":"示例"
}
```
# 说明：
根据用户对弹窗的操作进行相应设置
