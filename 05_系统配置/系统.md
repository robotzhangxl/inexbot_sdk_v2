## 时间设置

> 上位机发送时间查询
### 命令字：0x1050

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| "type" | int | 1: 获取系统开始时间至现在秒数，0: 现实时间，double 类型 |
| "format" | int | 1: 获取北京时间，0: 获取时间戳，double 类型 |

```json
{
"type": 0,
"format": 0
}
```

### 命令字：0x1051

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| "tType" | int | 1: 获取系统开始时间至现在秒数，0: 现实时间；int 类型 |
| "format" | int | 1: 获取北京时间，0: 获取时间戳；int 类型 |
| "date" | object | 包含日期和时间信息，分别记录年、月、日、时、分、秒、毫秒等值，通常用于表示某一时刻的完整时间信息。 |
| "year" | int | 日期中的年份；int 类型；mon":1；日期中的月份；int 类型 |
| "day" | int | 日期中的天；int 类型 |
| "hour" | int | 时间中的小时；int 类型 |
| "min" | int | 时间中的分钟；int 类型 |
| "sec" | object | 时间中的秒数；int 类型 |
| "msec" | int | 时间中的毫秒数；int 类型 |
| "timestamp" | object | 以秒和纳秒的形式记录时间戳，用于高精度的时间计算，通常与日期结合使用。；format为0时显示。 |
| "s" | int | 时间戳中的秒数；int 类型 |
| "ns" | int | 时间戳中的纳秒数；int 类型 |

```json
{
"tType": 0,
"format": 0,
"date": {
"year": 2023,
"mon": 1,
"day": 1,
"hour": 1,
"min": 1,
"sec": 1,
"msec": 1
},
"timestamp": {
"s": 1,
"ns": 1
}
}
```
