# EIP

EtherNet/IP 协议配置，用于与控制器进行数据交换。

---

## 概述

EIP（EtherNet/IP）是基于 Ethernet 的工业通讯协议，支持与控制器进行高速数据读写。适用于需要与 PLC 或其他工业设备进行数据交换的场景。

### 数据类型说明

| 类型 | 变量类型 | 绑定类型 | 说明 |
|------|----------|----------|------|
| `0` | 全局 bool 变量 | 全局 bool 绑定变量 | 布尔值（输入/输出） |
| `1` | 全局 int 变量 | 全局 int 绑定变量 | 整数值 |
| `2` | 全局 double 变量 | 全局 double 绑定变量 | 双精度浮点值 |

### IO 点数说明

| type 值 | IO 点数 | 说明 |
|---------|---------|------|
| `1` | 1点 | 1个输入 + 1个输出 |
| `4` | 4点 | 4个输入 + 4个输出 |
| `8` | 8点 | 8个输入 + 8个输出 |
| `12` | 12点 | 12个输入 + 12个输出 |
| `16` | 16点 | 16个输入 + 16个输出 |

### 编号说明

| 编号范围 | 类型 | 说明 |
|----------|------|------|
| `1` ~ `10` | 输出 | 写入控制器的数据 |
| `11` ~ `20` | 输入 | 从控制器读取的数据 |

## 配置流程

### 步骤一：设置 EIP 参数

配置 IO 点数、变量绑定方式、数据存储位置等。

**命令字：** `0x7360`

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| type | int | 是 | IO 点数配置：`1` / `4` / `8` / `12` / `16` |
| grouptype | int | 是 | 组号方式：`0`=手填，`1`=全局 int 类型变量，`2`=全局 int 类型绑定变量 |
| group | int | 是 | 组号中的数字（配合 grouptype 使用） |
| addtype | int | 是 | 数值存入方式：<br>`0`=全局 int 变量<br>`1`=全局 int 绑定变量<br>`2`=全局 double 变量<br>`3`=全局 double 绑定变量 |
| add | int | 是 | 数值存入的数字（配合 addtype 使用） |
| num | int | 是 | 编号：`1`~`10`=输出，`11`~`20`=输入 |
| value | any | 否 | 要写入的值 |
| able | bool | 否 | 使能开关：`true`=使能，`false`=不使能 |

### 请求示例

```json
{
  "type": 4,
  "grouptype": 1,
  "group": 3,
  "addtype": 2,
  "num": 5,
  "value": 1,
  "able": true
}
```

> **示例说明：** 配置4点EIP，组号从全局int变量读取，组号为3，数据存入全局double变量add位置。

---

### 步骤二：设置连接参数

配置网口、读写数据长度、扫描周期等连接相关参数。

**命令字：** `0x7363`

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| enable | bool | 是 | 连接开关：`true`=打开连接，`false`=关闭连接 |
| scancycle | int | 否 | 扫描周期（ms） |
| outtimecycle | int | 否 | 超时周期（ms） |
| writenum | int | 否 | 写入数据长度，范围 [0, 256] |
| readnum | int | 否 | 读取数据长度，范围 [0, 256] |
| networkname | string | 否 | 网口名称，如 `"eth0"` |
| writetype | int | 否 | 本机写数据类型：`0`=全局 bool 变量，`1`=全局 bool 绑定变量 |
| writeadd | int | 否 | 本机写数据的位置 |
| readtype | int | 否 | 本机读数据类型：`0`=全局 bool 变量，`1`=全局 bool 绑定变量 |
| readadd | int | 否 | 本机读数据的位置 |

### 请求示例

```json
{
  "enable": true,
  "writenum": 34,
  "readnum": 45,
  "scancycle": 100,
  "outtimecycle": 30,
  "networkname": "eth0",
  "writetype": 0,
  "writeadd": 1,
  "readtype": 0,
  "readadd": 0
}
```

## 查询命令

### 上位机获取 EIP 参数配置

**命令字：** `0x7361`

> 无需请求参数，响应格式同 `0x7360`。

---

### 上位机获取连接参数配置

**命令字：** `0x7364`

> 无需请求参数，响应格式同 `0x7363`。

---

### 上位机获取连接状态

**命令字：** `0x7366`

> 无需请求参数。用于检查 EIP 连接是否正常建立。

---

### 上位机获取读取数据状态

**命令字：** `0x7366`

**响应参数：**

| 参数 | 类型 | 说明 |
|------|------|------|
| num | int | 编号：`0`~`9`=输出，`10`~`19`=输入 |
| value | any | 当前值 |

### 响应示例

```json
{
  "num": 2,
  "value": 1
}
```

## 数据同步

### 设置 EIP 数据变化同步

**命令字：** `0x7369`

> 控制当 EIP 数据变化时，绑定的全局变量是否同步更新。

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| state | bool | 是 | `true`=EIP 数据变化时，参数同步更新<br>`false`=参数保持不变 |

### 请求示例

```json
{
  "state": true
}
```
