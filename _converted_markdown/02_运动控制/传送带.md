# 传送带

> 上位机发送，轨道传送位置设置

---

## 命令字：0x8200

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| "conveyorID" | int | 传送带工艺号；int类型；取值范围[1,9] |
| "position" | object |  |
| "receLatestPos" | double | 最迟接收X值；double类型 |
| "trackRangeXMax" | double | 跟踪最大范围；double类型 |
| "trackStartXPoint" | double | 跟踪开始X点；double类型 |
| "robot" | int | 机器人号；int类型；取值范围[1,4] |

```json
{
"conveyorID":1,
"position":
{
"receLatestPos":0.0,	//最迟接收X值
"trackRangeXMax":0.0,	//跟踪最大范围
"trackStartXPoint":0.0	//跟踪开始X点
},
"robot":1
}
```
> 上位机发送，轨道传送位置查询
