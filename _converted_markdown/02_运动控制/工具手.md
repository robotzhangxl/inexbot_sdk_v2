# 工具手

工具坐标系配置，包括工具参数设置、工具号切换和工具标定。

---

## 工具手参数

### 上位机设置工具手参数

**命令字：** `0x3500`

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| robot | int | 是 | 当前机器人号，范围 [1, 4] |
| tool | int | 是 | 工具手号，范围 [1, 999] |
| parameter | object | 是 | 弧度制工具参数 |
| note | string | 否 | 工具手注释 |

#### parameter 参数

| 参数 | 类型 | 说明 |
|------|------|------|
| A | double | 绕 A 轴旋转（弧度制范围 [-3.1416, 3.1416]，角度制 [-180, 180]） |
| B | double | 绕 B 轴旋转 |
| C | double | 绕 C 轴旋转 |
| X | double | X 轴偏移（mm），范围 [-2000, 2000] |
| Y | double | Y 轴偏移（mm） |
| Z | double | Z 轴偏移（mm） |
| payloadInertia | double | 负载惯量（0.001 kgm²） |
| payloadMass | double | 负载质量（kg） |
| payloadMassCenterX | double | 负载质心 X（m） |
| payloadMassCenterY | double | 负载质心 Y（m） |
| payloadMassCenterZ | double | 负载质心 Z（m） |

### 请求示例

```json
{
  "robot": 1,
  "tool": 1,
  "parameter": {
    "A": 0.0,
    "B": 0.0,
    "C": 0.0,
    "X": -68.0,
    "Y": 3.0,
    "Z": 162.0,
    "payloadInertia": 0.0,
    "payloadMass": 3.0,
    "payloadMassCenterX": 0.0,
    "payloadMassCenterY": 0.0,
    "payloadMassCenterZ": 0.170
  },
  "note": ""
}
```

### 上位机查询工具手参数

**命令字：** `0x3501`

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| robot | int | 是 | 当前机器人号 |
| tool | int | 是 | 工具手号 |

### 响应参数

> 同设置参数，额外包含 `parameterDeg`（角度制参数）

## 工具号

### 上位机请求切换工具号

**命令字：** `0x3510`

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| robot | int | 是 | 当前机器人号 |
| tool | int | 是 | 目标工具手号，`0`=无工具手 |

---

### 上位机查询当前工具手号

**命令字：** `0x3511`

### 响应参数

| 参数 | 类型 | 说明 |
|------|------|------|
| robot | int | 当前机器人号 |
| curToolNum | int | 当前已选中工具手号 |
| tool | int | 当前工具手号 |

## 工具标定

六点法（6个点）和七点法（7个点）标定工具坐标系。

> **流程：** 先用 `0x3521` 标定每个点，所有点标定完成后用 `0x3520` 计算标定结果。

### 上位机请求标定计算

**命令字：** `0x3520`

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| robot | int | 是 | 当前机器人号 |
| tool | int | 是 | 工具手编号 |
| calibrationType | int | 是 | 标定方法：`6`=六点法，`7`=七点法 |

---

### 上位机标定 N 号点

**命令字：** `0x3521`

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| robot | int | 是 | 当前机器人号 |
| tool | int | 是 | 工具手编号 |
| calibrationType | int | 是 | 标定方法：`6`=六点法，`7`=七点法 |
| point | int | 是 | 标定点编号（六点法：[0,5]，七点法：[0,6]） |

---

### 上位机查询 N 号点位置信息

**命令字：** `0x3524`

### 响应参数

| 参数 | 类型 | 说明 |
|------|------|------|
| robot | int | 当前机器人号 |
| calibrationType | int | 标定方法 |
| point | int | 标定点编号 |
| position | double[] | 标定点直角坐标信息列表 |

---

### 上位机查询全部点标定状态

**命令字：** `0x3526`

### 响应参数

| 参数 | 类型 | 说明 |
|------|------|------|
| robot | int | 当前机器人号 |
| calibrationType | int | 标定方法 |
| status | int[] | 各点标定状态列表（`1`=已标定，`0`=未标定） |

---

### 上位机请求清除 N 号点标定状态

**命令字：** `0x3528`

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| robot | int | 是 | 当前机器人号 |
| tool | int | 是 | 工具手编号 |
| calibrationType | int | 是 | 标定方法 |
| point | int | 是 | 标定点编号 |
