# 干涉区

空间立方体干涉区配置，进入或离开区域时触发 IO 输出或禁止运动。

---

## 干涉区范围

### 上位机设置干涉区范围

**命令字：** `0x3800`

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| robot | int | 是 | 当前机器人号，范围 [1, 4] |
| num | int | 是 | 工艺号，范围 [1, 99] |
| enable | bool | 是 | 干涉区使能 |
| isInInterfer | int | 是 | 区域判断：`0`=在区域外，`1`=在区域内 |
| minX / maxX | double | 是 | X 轴范围（mm），[-9999, 9999] |
| minY / maxY | double | 是 | Y 轴范围（mm），[-9999, 9999] |
| minZ / maxZ | double | 是 | Z 轴范围（mm），[-9999, 9999] |
| varName | string | 否 | 输出变量（GB001-GB999） |
| port | int | 否 | 输出 IO 端口 |
| value | int | 否 | 输出值：`0` 或 `1` |
| workType | int | 否 | 作用方式：`0`=状态输出，`1`=禁止运动 |

---

### 上位机查询干涉区范围

**命令字：** `0x3801`

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| robot | int | 是 | 当前机器人号 |
| num | int | 是 | 工艺号 |

### 响应参数

| 参数 | 类型 | 说明 |
|------|------|------|
| robot | int | 当前机器人号 |
| num | int | 工艺号 |
| param | object | 干涉区参数 |
| calibrationState1 | int | 标记点1标定状态（`0`=未标定，`1`=已标定） |
| calibrationState2 | int | 标记点2标定状态 |

---

## 干涉点

通过标记两个空间点来定义干涉区。

### 上位机请求标记干涉点

**命令字：** `0x3810`

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| robot | int | 是 | 当前机器人号 |
| num | int | 是 | 工艺号 |
| pos | int | 是 | 标记点号：`1`=点1，`2`=点2 |
| calibrationState1 | int | 是 | 点1状态：`0`=标定，`1`=取消 |
| calibrationState2 | int | 是 | 点2状态：`0`=标定，`1`=取消 |

---

### 上位机请求标记完成计算

**命令字：** `0x3811`

### 上位机请求移动到标记点

**命令字：** `0x3812`

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| robot | int | 是 | 当前机器人号 |
| num | int | 是 | 工艺号 |
| pos | int | 是 | 目标点号：`1`=点1，`2`=点2 |
