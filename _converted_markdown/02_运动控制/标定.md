# 标定

机器人标定方法，包括四点标定、20点标定和外部轴标定。

---

## 四点标定

机器人末梢移动到四个标记点进行标定，计算杆长度和轴零点偏移，结果可填入 DH 参数。

### 上位机四点标定计算请求

**命令字：** `0x3700`

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| ParamOf4Points | double[] | 是 | [L1, L2] 长度（mm）：末梢从 A 点到 C 点的 X/Y 移动距离 |

---

### 上位机 N 号点标定请求

**命令字：** `0x3701`

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| pointNum | int | 是 | 标定点编号，范围 [0, 3] |
| status | int | 是 | 标定状态：`0`=取消标定，`1`=标定 |

---

### 上位机查询标定信息

**命令字：** `0x3706`

### 响应参数

| 参数 | 类型 | 说明 |
|------|------|------|
| result | bool | 计算是否成功 |
| dtheta | double[] | 轴零点偏移列表（长度=2，`null`=无结果） |
| length | double[] | 杆长度列表（长度=2，`null`=无结果） |
| status | int[] | 各点标定状态（`0`=未标定，`1`=已标定） |

---

### 上位机请求将计算结果填入 DH 参数

**命令字：** `0x3704`

---

## 20点标定

通过 20个点（或 21点、15点、12点、2点）进行机器人校准。

> 标定结果 > 10mm 时请重新标定。

### 支持的标定方法

| 方法 | 说明 |
|------|------|
| 2点法 | 2个点 |
| 12点法 | 12个点 |
| 15点法 | 15个点 |
| 20点法 | 20个点，有校准零点 |
| 21点法 | 21个点，无校准零点 |

### 上位机标定计算请求

**命令字：** `0x3710`

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| robot | int | 是 | 当前机器人号，范围 [1, 4] |
| tool | int | 是 | 工具手编号，范围 [1, 999] |
| calibrationType | int | 是 | 标定方法：2, 12, 15, 20, 21 |
| noCalZero | bool | 否 | 是否校准零点 |

---

### 上位机标定 N 号点请求

**命令字：** `0x3711`

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| robot | int | 是 | 当前机器人号 |
| tool | int | 是 | 工具手编号 |
| calibrationType | int | 是 | 标定方法 |
| point | int | 是 | 标定点编号（根据方法不同范围不同） |

---

### 上位机查询全部点标定状态

**命令字：** `0x3716`

### 响应参数

| 参数 | 类型 | 说明 |
|------|------|------|
| robot | int | 当前机器人号 |
| calibrationType | int | 标定方法 |
| status | int[] | 各点标定状态（`1`=已标定，`0`=未标定） |

---

## 外部轴标定

旋转轴和直线轴（地轨）的协同标定。

### 上位机外部旋转轴标定计算请求

**命令字：** `0x3730`

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| robot | int | 是 | 当前机器人号 |
| calibrateNum | int | 是 | 轴组号，范围 [1, 3] |

---

### 上位机设置地轨协同参数

**命令字：** `0x3739`

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| calibrateResult | bool | 是 | 地轨是否协同 |
| conversionRatio | double[] | 是 | 轴转换比列表（长度=3） |
| trackMap | int[] | 是 | 地轨各轴方向（长度=3）：`0`=未选择，`1`=X，`2`=Y，`3`=Z |

---

### 上位机查询地轨方向校准系数

**命令字：** `0x3740`（推测）

> `direction_ratio`：方向校准系数 [X, Y, Z] 方向系数
