# 用户坐标

用户坐标系标定与切换。

---

## 用户坐标标定

### 上位机请求用户坐标标定计算

**命令字：** `0x3560`

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| robot | int | 是 | 当前机器人号，范围 [1, 4] |
| user | int | 是 | 用户坐标号，范围 [1, 999] |

---

### 上位机请求标记用户坐标原点/X值/Y值

**命令字：** `0x3562`

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| robot | int | 是 | 当前机器人号 |
| user | int | 是 | 用户坐标号 |
| require | string | 是 | 标记类型：`0`=原点，`X`=X 值，`Y`=Y 值，`OXY`=标记所有点 |
| posZero | double[] | 否 | 原点位置信息列表（require=OXY 时存在） |
| posX | double[] | 否 | X 值位置信息列表 |
| posY | double[] | 否 | Y 值位置信息列表 |

### 响应参数

| 参数 | 类型 | 说明 |
|------|------|------|
| robot | int | 当前机器人号 |
| user | int | 用户坐标号 |
| require | string | 标记类型 |
| status | bool | 标定状态：`true`=已标定，`false`=未标定 |
| position | double[] | 弧度制位置信息列表 |
| positionDeg | double[] | 角度制位置信息列表 |

---

### 上位机查询用户坐标标定点位置信息

**命令字：** `0x3570`

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| robot | int | 是 | 当前机器人号 |
| user | int | 是 | 用户坐标号 |
| require | string | 是 | 查询类型：`0`=原点，`X`=X 值，`Y`=Y 值，`Calibration`=计算后位置信息 |

**命令字：** `0x3593`
