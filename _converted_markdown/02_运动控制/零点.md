# 零点

零点偏移、单圈/多圈值、编码器状态等参数配置。

---

## 上位机请求标记无间隙方向

**命令字：** `0x3600`

---

## 上位机请求清除轴多圈值

**命令字：** `0x3601`

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| robot | int | 是 | 当前机器人号，范围 [1, 4] |
| axis | int | 是 | 轴号：`0`=清除所有轴，`1`~`7`=指定轴 |
| clear | bool | 是 | 是否清除多圈值 |

### 响应参数

| 参数 | 类型 | 说明 |
|------|------|------|
| robot | int | 当前机器人号 |
| axis | int | 被清除的轴 |
| result | int | 清除结果：`0`=失败，`1`=成功 |

---

## 上位机查询编码器状态

**命令字：** `0x3603`

### 响应参数

| 参数 | 类型 | 说明 |
|------|------|------|
| robot | int | 当前机器人号 |
| encoderUndervoltage | bool[] | 各轴欠压报警列表（长度=机器人轴数） |
| encoderUnderVoltageExternal | bool[] | 外部轴欠压报警列表 |

---

## 上位机设置零点偏移

**命令字：** `0x3605`

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| robot | int | 是 | 当前机器人号 |
| axis | int | 是 | 轴号：`0`=设置全部轴，`1`~`7`=指定轴 |
| mode | int | 是 | 偏移方式：`0`=度数，`1`=偏移量 |
| value | double[] | 是 | 零点偏移值列表（长度=7） |

> 度数范围：[-360, 360]

---

## 上位机设置单圈值

**命令字：** `0x3606`

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| robot | int | 是 | 当前机器人号 |
| value | int[] | 是 | 各轴单圈值列表（长度=6），范围 [-2^31, 2^31-1] |

---

## 上位机查询单圈值

**命令字：** `0x3607`

### 响应参数

| 参数 | 类型 | 说明 |
|------|------|------|
| robot | int | 当前机器人号 |
| value | int[] | 各轴单圈值列表 |

---

## 上位机查询多圈值（零点偏移为编码器时）

**命令字：** `0x330B`

### 响应参数

| 参数 | 类型 | 说明 |
|------|------|------|
| robot | int | 当前机器人号 |
| value | int[] | 电机编码器值列表 |
