| "weld_finish_time" | string | 焊接完成回抽时间；double 类型；取值范围[0,5]s |
| "break_back_enable" | string | 断弧回抽使能；bool类型；true: 开启；false；关闭 |
| "weld_finish_function" | string | 焊接完成回抽功能；bool类型；true: 开启；false；关闭 |
| "simulation_set_zero" | string | 灭弧模拟量置零功能；bool类型；true: 开启；false；关闭 |

## 电流电压


### 电流电压匹配设置

### 命令字： 0x8310

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| "current_control" | object |  |
| "set_voltage" | array | 设置电流；double类型；取值范围：[0,10]A |
| "actual_current" | array | 实际电流；double类型；取值范围：[0,999]A；voltage_control": |
| "set_voltage" | array | 设置电压；double类型；取值范围：[0,10]V |
| "actual_voltage" | array | 实际电压；double 类型；取值范围：[0,999]V |
| "robot" | int | 当前机器人；int型；取值范围[1,4] |

{
"current_control":
{

"actual_current":[1,2,3,4,5,6,7,8],
},
"voltage_control":
{

"actual_voltage":[1,2,3,4,5,6,7,8],
}
"robot":1
}

### 控制器回复电流电压匹配

### 命令字： 0x8312

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| "set_voltage" | array | 设置电流；int 类型；取值范围：[0,10]A |
| "actual_current" | array | 实际电流；int 类型；取值范围：[0,999]A |

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| "num" | string | 工艺号；int 类型；从1开始，可以设置 1-99个不同焊接工艺号 |
| "notes" | string | 注释；string类型 |
| "arcoff_parameters" | object | 收弧参数 |

| "arcoff_voltage" | double | 收弧电压；double 类型；取值范围[-1000,1000]V |
| "arcoff_current" | string | 收弧电流；double 类型；取值范围[0,1000]A |
| "arcoff_time" | double | 收弧时间；double 类型；取值范围[0,5]s |
