# 码垛工艺
## 抓手
# 说明
抓手设置就是设置码垛过程中用到的工具手。两个吸盘分别取料（一个吸盘吸完后再切换到另一个吸盘取料），那么要设置成两个抓手；两个吸盘同时取料，则设置成一个抓手；如果两个吸盘分别放料（一个吸盘放完后切换到另一个吸盘放料），此时需要设置两个抓手；两个吸盘同时放料，则设置成一个抓手。
# 上位机设置抓手参数
命令字：0x8000
"robot":1
当前机器人；int类型
取值范围：[1,4]
"craftID":1
工艺号；int类型
"pallet":
"gripperNum":4,
抓手数目；int类型
取值范围：[1,4]
"gripper":[1,3,4,1]
抓手工具号；int类型
第i个数代表抓手i的工具号
```json
{
"robot":1,
"craftID":1,
{
"gripperNum":2,
}
```
