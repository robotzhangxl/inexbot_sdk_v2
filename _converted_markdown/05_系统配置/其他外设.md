# 其他外设


## 编码器

> 说明
> 上位机设置外部轴编码器溢值参数
### 命令字：0x7200

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| "externalAxis" | object | 外部轴编码器最大值最小值参数列表，长度为当前外部轴轴数 |
| "encode" | object | 编码器最大值最小值参数列表，长度为该轴从动轴数目+1 |
| "max" | string | 编码器最大值；double类型，范围[-2^32,2^32] |
| "min" | string | 编码器最小值；double类型，范围[-2^32,2^32] |
| "externalGroupNum" | int | 要设置参数的轴组号；int类型，范围[1,3] |
| "robot" | int | 当前机器人号；int类型，范围[1,4] |

```json
{
"externalAxis":
[
{
"encode":
[
{
"max":"100.0",
"min":"-100.0"
}
]
}
],
"externalGroupNum":1,
"robot":2
}
```

### 命令字：0x7201

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| "externalGroupNum" | int | 要查询参数的轴组号；int类型，范围[1,3] |
| "robot" | int | 当前机器人号；int类型，范围[1,4] |

```json
{
"externalGroupNum":1,
"robot":2
}
```

### 命令字：0x7202

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| "externalAxis" | object | 外部轴编码器最大值最小值参数列表，长度为当前外部轴轴数 |
| "enable" | int | 编码器多圈值溢出计数功能；int类型，范围[0,1]；0表示未开启，1表示已开始 |
| "encode" | object | 编码器最大值最小值参数列表，长度为该轴从动轴数目+1 |
| "max" | string | 编码器最大值；double类型，范围[-2^32,2^32] |
| "min" | string | 编码器最小值；double类型，范围[-2^32,2^32] |
| "externalGroupNum" | int | 轴组号；int类型，范围[1,3] |
| "robot" | int | 当前机器人号；int类型，范围[1,4] |

```json
{
"externalAxis":
[
{
"enable":0,
"encode":
[
{
"max":"100.0",
"min":"-100.0"
}
]
}
],
"externalGroupNum":1,
"robot":2
}
```

## 伺服内部参数

> 说明
> 控制器发送伺服内部参数时会十个参数为一组，分多次发送给上位机

## PWM

> 说明
> 上位机设置PWM参数
### 命令字：0x7260

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| "duty" | int | pwm占空比；double类型 |
| "freq" | int | pwm频率；double类型 |
| "pwmOutTimeMs" | int | pwm输出时间；UInt类型，范围[1000-5000]ms；输入0的时候则一直输出pwm |
| "pwmPort" | int | pwm端口；UInt类型 |

```json
{
"duty":50,
"freq":1000,
"pwmOutTimeMs":1000,
"pwmPort":1
}
```

### 命令字：0x7261

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| "pwmPort" | int | pwm端口；UInt类型 |

```json
{
"pwmPort":1
}
```

### 命令字：0x7262

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| "duty" | int | pwm占空比；double类型 |
| "freq" | int | pwm频率；double类型 |
| "pwmPort" | int | pwm端口；UInt类型 |

```json
{
"duty":50,
"freq":1000,
"pwmPort":1
}
```

### 命令字：0x7263

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| "ok" | bool | 输出时间结果；bool类型 |

```json
{
"ok":true
}
```
