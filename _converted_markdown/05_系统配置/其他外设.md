# 其他外设
### 编码器
说明
上位机设置外部轴编码器溢值参数
命令字：0x7200
"externalAxis":
外部轴编码器最大值最小值参数列表，长度为当前外部轴轴数
"encode":
编码器最大值最小值参数列表，长度为该轴从动轴数目+1
"max":"100.0"
编码器最大值；double类型，范围[-2^32,2^32]
"min":"-100.0"
编码器最小值；double类型，范围[-2^32,2^32]
"externalGroupNum":1
要设置参数的轴组号；int类型，范围[1,3]
"robot":2
当前机器人号；int类型，范围[1,4]
```json
{
[
{
[
{
"max":"100.0",
"min":"-100.0"
}
]
}
"externalGroupNum":1,
}
```
上位机查询外部轴编码器溢值参数
命令字：0x7201
"externalGroupNum":1
要查询参数的轴组号；int类型，范围[1,3]
"robot":2
当前机器人号；int类型，范围[1,4]
```json
{
"externalGroupNum":1,
}
```
控制器回复上位机外部轴编码器溢值参数
命令字：0x7202
"externalAxis":
外部轴编码器最大值最小值参数列表，长度为当前外部轴轴数
"enable":0
编码器多圈值溢出计数功能；int类型，范围[0,1]
0表示未开启，1表示已开始
"encode":
编码器最大值最小值参数列表，长度为该轴从动轴数目+1
"max":"100.0"
编码器最大值；double类型，范围[-2^32,2^32]
"min":"-100.0"
编码器最小值；double类型，范围[-2^32,2^32]
"externalGroupNum":1
轴组号；int类型，范围[1,3]
"robot":2
当前机器人号；int类型，范围[1,4]
```json
{
[
{
"enable":0,
[
{
"max":"100.0",
"min":"-100.0"
}
]
}
"externalGroupNum":1,
}
```

### 外部接口
说明

"canPortSum":1

"encodorPortSum":2

"pwmPortSum":4

```json
{
"canPortSum":1,
"encodorPortSum":2,
}
```

### 伺服内部参数
说明
控制器发送伺服内部参数时会十个参数为一组，分多次发送给上位机
上位机设置伺服参数
命令字：0x7240
"robot":1
当前机器人号；int类型，范围[1,4]
"servo":
伺服内部参数列表；string类型
不同机型获取到参数不一致
"位置指令低通滤波器截止频率":
参数说明；string类型
"value":100.0
设置的参数值；int类型
"servoNum":1
需要设置的轴号；int类型
"temporary_save":0
是否为临时储存；int类型，范围[0,1]
0表示非临时储存，1表示临时储存
```json
{
"robot":1,
{
}
"servoNum":1,
}
```
上位机查询伺服内部参数
命令字：0x7241
"robot":1
当前机器人号；int类型，范围[1,4]
"servoNum":1
需要设置的轴号；int类型
```json
{
"robot":1,
}
```
控制器回复上位机伺服内部参数
命令字：0x7242
"robot":1
当前机器人号；int类型，范围[1,4]
"sendNum":10
已发送参数数量；int类型
"servo":
伺服内部参数列表；string类型
不同机型获取到参数不一致
"位置指令低通滤波器截止频率":
参数说明；string类型
"hex":false
是否为十六进制；bool类型
"index":8299
地址；int类型
"lowLimit":1
最小限制；int类型
"unit":"0.1HZ"
单位；string类型
"upLimit":1000
最大限制；int类型
"value":100.0
参数值；int类型
"servoNum":1
查询的轴号；int类型
"totalNum":38
内部参数的总数量；int类型
```json
{
"robot":1,
{
"hex":false,
"lowLimit":1,
"unit":"0.1HZ",
{
"hex":false,
"lowLimit":0,
"unit":"‰",

"hex":true,
"lowLimit":0,
"unit":"",
{

### PWM
说明

"freq":1000
pwm频率；double类型
"pwmOutTimeMs":1000
pwm输出时间；UInt类型，范围[1000-5000]ms
输入0的时候则一直输出pwm
"pwmPort":1
pwm端口；UInt类型

{
}
```

"freq":1000

}
